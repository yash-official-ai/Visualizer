# 📊 Data Analysis & Visualization Program

A menu-driven **Python project** built using **Object-Oriented Programming (OOP)**, **Pandas**, **Matplotlib**, and **Seaborn** to analyze and visualize sales data from CSV files.

---

## ✨ Features

- 📂 Load CSV Dataset
- 🔍 Explore Dataset
  - First 5 Rows
  - Last 5 Rows
  - Column Names
  - Data Types
  - Dataset Info
- 🛠️ Perform DataFrame Operations
- 🧹 Handle Missing Values
  - Display Missing Values
  - Fill with Mean
  - Drop Missing Values
  - Replace Missing Values
- 📈 Generate Descriptive Statistics
- 📊 Create Visualizations
  - Bar Plot
  - Line Plot
  - Scatter Plot
  - Pie Chart
  - Histogram
  - Stack Plot
- 💾 Save Visualization
- ❌ Exit Program

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- OOP (Object-Oriented Programming)

---

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

2. Run the program:

```bash
python main.py
```

3. Enter the CSV file path when prompted.
